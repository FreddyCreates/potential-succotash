## Running the Examples

Each example is a self-contained Node.js module with a demonstration:

```bash

node examples/academic-sticks/01-neurochemistry-ode.js

# Run Kuramoto Oscillator emergence demo
node examples/academic-sticks/02-kuramoto-oscillator.js

# Run Phi Mathematics exploration

# Run Memory Palace (Phi-Encoded Spatial Memory)

# Run NeuroCore Stack demo
```
