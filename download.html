🔧 How to Install

1

Download

Download individual extension .zip files or the full bundle above.

2

Unzip

Extract each .zip to its own folder. Each folder contains manifest.json, background.js, content.js, and icons.

3

Open Chrome Extensions

Navigate to chrome://extensions in Chrome, Edge, or Brave. Enable Developer mode (toggle in top-right).

4

Load Extension

Click "Load unpacked" and select the unzipped extension folder. The extension is now live and running 24/7.

⚡ Quick Install (Windows)

1

Download Bundle

Download all-extensions.zip and extract to C:\organism-extensions\

2

Run Installer

Open PowerShell as Admin and run: .\install-extensions.ps1 — it opens Chrome with all extensions loaded.
